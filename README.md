# davidserret.github.io

Personal portfolio site — [davidserret.github.io](https://davidserret.github.io)

## Deploy

This is a static site with no build step.

1. Push `index.html` to the `main` branch of this repo
2. Go to repo Settings → Pages → Source: Deploy from branch → main → / (root)
3. Site is live at https://davidserret.github.io in ~60 seconds

## Add avatar

1. Take a photo (good light, neutral background)
2. Save as `avatar.jpg` in the repo root
3. Uncomment the `<img>` tag in `index.html` under the hero section
